EmbedGoogleMap

EmbedGoogleMap class

A widget that embeds a Google Map.

This widget allows you to embed a Google Map using the provided parameters. You can customize the appearance of the map by specifying the backgroundColor. The loadingProgress callback allows you to track the loading progress of the map. The onError callback is triggered when an error occurs while loading the map. The onNavigationRequest callback is called when a navigation request is made within the map.

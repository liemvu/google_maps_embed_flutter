OnNavigateRequest typedef

OnNavigateRequest = FutureOr<NavigationDecision> Function(NavigationRequest request)

A typedef representing a callback function that handles navigation requests.

The OnNavigateRequest function takes a NavigationDecision as a parameter and returns a Future or FutureOr of NavigationDecision. It is used to determine whether to allow or cancel a navigation request.
